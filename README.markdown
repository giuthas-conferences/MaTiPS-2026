# Beamer poster template

My personal beamer poster template that I've modified from something I
got from a colleague years ago. Not supported in any way. I will not
accept pull requests. Use as you will.

## Getting Started

This is how:

* Clone the repo.
* ALWAYS make a new repo for the poster you are making. This way you won't
  clutter the template with nonsense.
* If the template needs updates, make them in the master branch and push to
  github.

### Prerequisites

The template uses my .bib exports from zotero. Just replace them with your own
files.
